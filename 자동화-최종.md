# 🚀 자동화 흐름 — Google Apps Script + GitHub Issues + Hermes

## 전체 파이프라인

```
누군가 구글폼 제출
    ↓
Google Apps Script 실행 (onFormSubmit)
    ↓
GitHub Issue 자동 생성 (flffkaos-pixel/ai-lab-free-services)
    ↓
GitHub Actions가 5분마다 Issues 확인
    ↓
신규 Issue 발견 → check-applications.py가 stdout에 출력
    ↓
Hermes CronJob이 auto-review-pipeline Skill로 처리
    ↓
논문 분석 or 영업 시뮬 → 결과를 Issue 댓글로 회신
```

## 장점

| 항목 | 설명 |
|------|------|
| 💰 비용 | 완전 무료 (Google Apps Script + GitHub Actions 모두 무료) |
| 🏦 카드 | 불필요 |
| 🖥️ PC 의존 | 없음 — 구글 서버 + 깃헙 서버에서 실행 |
| ⚡ 속도 | 구글폼 제출 → 10분 내 처리 |

## 설정 순서

### 1. GitHub Personal Access Token 생성 (무료)
https://github.com/settings/tokens → Generate new token (classic)
- 체크: `repo`
- 생성 후 토큰 복사

### 2. GitHub Secrets에 토큰 등록
https://github.com/flffkaos-pixel/ai-lab-free-services/settings/secrets/actions
- 이름: `GH_TOKEN`
- 값: 방금 복사한 토큰

### 3. 구글폼에 Apps Script 연결
구글폼 편집 → ⋮ → 스크립트 편집기 → 코드 붙여넣기 → 트리거 추가

### 4. 끝!
누군가 폼 작성 → GitHub Issue 생성 → Hermes가 자동 처리
