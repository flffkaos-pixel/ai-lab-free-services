# ♟️ B2B 영업 시뮬레이터 — 무료 서비스 빌드

## 💡 왜 이걸 만드는가?

영업 교육 시장은 2조원. 하지만 모든 교육이 **이론 강의**에서 끝난다.
- "SPIN 기법을 배웠다" → 실제 고객 앞에선 말문이 막힌다
- 역할극 강사는 1회 50~100만원, 1년에 몇 번 못 한다
- 영업 실패는 실제 매출 손실로 이어진다. 연습할 곳이 없다

**AI 영업 시뮬레이터는 진짜 바이어처럼 행동하는 AI와 무제한 무료 연습이다.**

---

## 🤖 사용할 AI 에이전트 8종

| # | 에이전트 | 역할 |
|---|----------|------|
| 1 | 🔍 Discovery Coach | SPIN/Gap Selling 기반 질문 → 가상 바이어 질문 생성 |
| 2 | ♟️ Deal Strategist | MEDDPICC 기준 반론 → 가상 바이어의 반대 논리 |
| 3 | 🏋️ Sales Coach | 실시간 피드백 → 영업사원의 답변 평가·개선점 제시 |
| 4 | 🧠 Behavioral Nudge Engine | 구매 심리 기반 압박 → 바이어의 감정적 반응 시뮬레이션 |
| 5 | 📊 Pipeline Analyst | 가상 파이프라인 점수화 → 3개월 후 예측 점수 |
| 6 | 🧠 Organizational Psychologist | 영업사원 심리 진단 → 자신감·협상 스트레스·소진 위험 |
| 7 | ⚔️ Strategy Duel Agent | 36계 기반 협상 전략 → 게임이론 기반 시나리오 |
| 8 | 📄 Document Generator | 세션 후 종합 성적표 PDF 생성 |

## 🎯 서비스 흐름

```
[영업사원이 업종/상품/난이도 선택]
         ↓
[AI 바이어가 역할 시작: "안녕하세요, 어떤 용무신지?"]
         ↓
[영업사원이 답변 → AI 바이어가 반응] ← SPIN/MEDDPICC 기반
         ↓
[세션 종료 후 AI 코치가 피드백]
         ↓
[종합 성적표 PDF + 개선 로드맵]
```

## 📋 시뮬레이션 시나리오 템플릿

### 업종별 프리셋
- **SaaS**: CTO 대상, 연간 계약 5천만원, 기존 경쟁사 사용 중
- **제조**: 구매팀장 대상, 단가 인하 압박, 기존 거래처 10년
- **컨설팅**: CEO 대상, 3개월 프로젝트 1억, RFP 경쟁 상황
- **의료기기**: 병원장 대상, 예산 부족, 식약처 인허가 복잡

### 난이도
- 🟢 초급: 정보 제공형 바이어, 예산 확보됨, 필요성 인지
- 🟡 중급: 경쟁사 비교 중, 가격 민감, 내부 의사결정 복잡
- 🔴 고급: 적대적, 숨은 의제 있음, 기존 솔루션 만족, 예산 없음

## 🖥️ Claude Code 명령어셋 — 영업 시뮬레이터

### 1단계: 시뮬레이션 설정
```
@discovery-coach @deal-strategist @behavioral-nudge-engine @strategy-duel-agent

지금부터 B2B 영업 시뮬레이션을 진행합니다.
당신들은 한 팀의 가상 바이어입니다.

[시나리오]
- 업종: SaaS (마케팅 자동화 툴)
- 대상: 중견기업 CMO (마케팅 총괄 임원)
- 상황: 현재 Salesforce 사용 중, 연간 5천만원 계약
- 숨은 니즈: 팀 생산성 저하 + 데이터 연동 문제
- 성격: 데이터 중심, ROI에 민감, 과거 SaaS 도입 실패 경험 있음

규칙:
1. 처음엔 가벼운 스몰토크로 시작
2. 영업사원이 질문할 때마다 SPIN 프레임워크에 기반해 반응
3. MEDDPICC 기준을 충족하는 답변이 나올 때까지 쉽게 설득되지 않음
4. Behavioral Nudge Engine이 구매 심리 단계를 시뮬레이션
5. Strategy Duel Agent가 협상 국면에서 36계 전략적 반응

시작하겠습니다: "안녕하세요, 만나서 반갑습니다. 오늘 저희에게 어떤 도움을 드릴 수 있을까요?"
```

### 2단계: 영업사원이 답변 → AI가 반응 (대화 반복)
> [영업사원의 실제 답변을 입력하면 AI 바이어가 반응. 이 과정을 5~10턴 반복]

### 3단계: 세션 종료 후 피드백
```
@sales-coach @pipeline-analyst @organizational-psychologist

방금 진행된 시뮬레이션 세션을 분석해줘.

[평가 영역]
1. SPIN 질문 품질: Situation → Problem → Implication → Need-Payoff 순서로 적절했는가
2. MEDDPICC 커버리지: Metrics, Economic Buyer, Decision Criteria, Paper Process, Implicate Pain, Champion, Competition — 각 요소를 얼마나 파악했는가
3. 반론 처리: 가격·경쟁사·기존 솔루션 반론에 얼마나 잘 대응했는가
4. 심리적 안정감: Organizational Psychologist가 영업사원의 자신감·스트레스 수준을 평가

점수: 각 항목 100점 만점
[종합 등급: S/A/B/C/D]
```

### 4단계: 성적표 PDF 생성
```
@document-generator 위 분석을 바탕으로
"B2B 영업 시뮬레이션 성적표" PDF를 생성해줘.

포맷:
- 표지: 영업사원 이름, 시나리오, 날짜
- 항목별 점수 (방사형 차트)
- 잘한 점 / 개선할 점
- 다음 세션 추천 시나리오
- 30일 개선 로드맵
```

## 💰 무료 서비스로 운영하는 방법

### 무료 모델
- **주 2회 무료 시뮬레이션** (난이도: 초급/중급)
- **프리미엄**: 주 5회 + 고급 시나리오 + 개인 코칭 플랜 → 월 5만원

### 타겟 커뮤니티
- **세일즈 커뮤니티**: "세일즈웍스", "세일즈맨", "영업의 신"
- **링크드인**: B2B 영업·SaaS 세일즈 그룹
- **스타트업 커뮤니티**: EO, 스타트업얼라이언스 — 초기 세일즈 교육 필요한 창업자
- **기업 교육 담당자**: 대기업 영업 교육팀에 무료 PoC 제안

### 구글폼 구성
```
1. 이름/회사/직무
2. 업종 선택 (SaaS/제조/컨설팅/의료/기타)
3. 희망 난이도
4. 특별히 연습하고 싶은 상황 (예: 가격 협상, 경쟁사 비교, 콜드콜)
5. 가능한 시간대 (AI 바이어가 24시간 준비되어 있으므로 유연함)
```
