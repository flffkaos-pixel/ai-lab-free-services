# 🚀 완전 초보자를 위한 AI 서비스 시작 가이드

## 1. Claude Code 설치 (10분)

### Claude Code가 뭔가요?
AI랑 대화하면서 일을 시킬 수 있는 프로그램이에요.
마치 똑똑한 직원 한 명 고용했다고 생각하면 됩니다.

### 설치 방법

**1) Node.js 설치**
- https://nodejs.org 들어가기
- 왼쪽에 있는 LTS 버전 다운로드
- 다운 받은 파일 더블클릭 → 다음다음다음

**2) Claude Code 설치**
- 키보드에서 `윈도우키 + R` 누르기
- `cmd` 입력하고 확인
- 까만 창에 아래 명령어 입력하고 엔터:

```
npm install -g @anthropic-ai/claude-code
```

**3) Claude 가입**
- https://claude.ai 들어가기
- 구글 계정으로 가입하기
- Pro 요금제 결제 (월 3만원 - 이거 하나면 끝!)

**4) 잘 설치됐나 확인하기**
- 까만 창에 `claude` 입력하고 엔터
- AI랑 대화할 수 있는 화면이 나오면 성공!

## 2. AI 직원들 설치하기 (5분)

### AI 직원이 뭔가요?
"agency-agents"라는 곳에서 만든 전문가 AI들이에요.
논문 전문가, 영업 전문가, 마케팅 전문가 같은 역할을 합니다.

### 설치 방법

까만 창(명령 프롬프트)에 아래 명령어를 한 줄씩 복사해서 엔터:

**1) 바탕화면에 폴더 만들기**
```
cd %USERPROFILE%\Desktop
mkdir ai-business
cd ai-business
```

**2) AI 직원 다운로드**
```
git clone https://github.com/msitarzewski/agency-agents.git
```
(이게 안 되면 https://github.com/msitarzewski/agency-agents 에서 초록색 "Code" 버튼 → Download ZIP → 압축 풀어서 ai-business 폴더에 넣기)

**3) Claude Code가 쓸 수 있게 직원 배치하기**
```
mkdir %USERPROFILE%\.claude\agents
copy agency-agents\academic\*.md %USERPROFILE%\.claude\agents\
copy agency-agents\sales\*.md %USERPROFILE%\.claude\agents\
copy agency-agents\specialized\specialized-model-qa.md %USERPROFILE%\.claude\agents\
copy agency-agents\specialized\language-translator.md %USERPROFILE%\.claude\agents\
copy agency-agents\specialized\specialized-document-generator.md %USERPROFILE%\.claude\agents\
copy agency-agents\product\product-behavioral-nudge-engine.md %USERPROFILE%\.claude\agents\
copy agency-agents\specialized\organizational-psychologist.md %USERPROFILE%\.claude\agents\
copy agency-agents\specialized\specialized-strategy-duel-agent.md %USERPROFILE%\.claude\agents\
```

다 됐어요! 이제 AI 직원들이 일할 준비가 끝났습니다.

## 3. 서비스 1: 논문 리뷰어 만들기

### 누구한테 팔까요?
대학원생, 교수, 연구원

### 어떻게 하나요?

**1) Claude Code 실행**
까만 창에 `claude` 입력하고 엔터

**2) 아래 글을 그대로 복사해서 붙여넣기**

```
@narratologist @psychologist @anthropologist @model-qa-specialist @language-translator

안녕하세요. 지금부터 논문 리뷰를 부탁드립니다.

아래 논문을 6가지 영역에서 분석해주세요:

1. 논리 구조: 주장이 논리적인가?
2. 가독성: 읽기 쉬운가?
3. 편향: 특정 문화/집단에 치우치지 않았나?
4. 방법론: 통계가 올바른가?
5. 영문 초록: 번역이 정확한가?
6. 종합 평가: 투고해도 될까?

문제점을 찾아주고, 어떻게 고칠지도 알려주세요.
마지막에 "이 논문의 투고 가능성은?" 이라고 평가해주세요.

[여기에 논문 내용을 붙여넣으세요]
```

**3) 결과가 나오면?**

나온 결과를 복사해서:
- 구글 문서(Google Docs)에 붙여넣기
- PDF로 저장
- 의뢰한 사람에게 이메일로 보내기

**이게 끝입니다.**

## 4. 서비스 2: 영업 시뮬레이터 만들기

### 누구한테 팔까요?
영업사원, 세일즈 교육 담당자, 창업자

### 어떻게 하나요?

**1) Claude Code 실행**
까만 창에 `claude` 입력하고 엔터

**2) 아래 글을 그대로 복사해서 붙여넣기**

```
@discovery-coach @deal-strategist @behavioral-nudge-engine @strategy-duel-agent

지금부터 B2B 영업 시뮬레이션을 시작합니다.

당신은 가상의 바이어입니다. 진짜 고객처럼 행동하세요.

[바이어 정보]
- 회사: 중견 IT기업
- 직급: 마케팅 총괄 임원 (CMO)
- 상황: 현재 다른 제품 사용 중, 연간 5천만원 계약
- 성격: 까다롭고 가격에 민감함

규칙:
- 제가 질문하기 전까지 니즈를 말하지 마세요
- 가격, 경쟁사, 도입 실패 경험 이야기를 최소 1번은 꺼내세요
- 너무 쉽게 설득되지 마세요
- 10턴 정도 대화가 오가면 "오늘 미팅은 여기까지" 하면서 끝내주세요

시작해주세요: "안녕하세요, 만나서 반갑습니다."
```

**3) AI 바이어랑 대화하기**

AI가 바이어 역할을 시작하면, 당신은 영업사원 역할로 답변하세요.
10턴 정도 대화한 후...

**4) 피드백 받기**

이어서 아래 글을 붙여넣으세요:

```
@sales-coach @pipeline-analyst @organizational-psychologist

방금 시뮬레이션에서 제 영업 스킬을 평가해주세요.

점수로 알려주세요:
- SPIN 질문은 잘했나요? (25점 만점)
- 고객 정보를 잘 파악했나요? (25점 만점)
- 고객 반론에 잘 대응했나요? (20점 만점)
- 관계는 잘 형성했나요? (15점 만점)
- 다음 단계를 잘 제안했나요? (15점 만점)

총점과 함께 잘한 점, 개선할 점을 알려주세요.
```

**이게 끝입니다.**

## 5. 돈은 어떻게 버나요?

### 무료로 시작하는 이유
처음부터 돈을 받으면 아무도 안 옵니다.
무료로 해주면서 "이런 서비스가 있구나" 알리는 게 먼저입니다.

### 무료 → 유료 전환 방법

**1) 첫 1~3개월: 완전 무료**
- "베타 테스트 중"이라고 말하기
- 일주일에 논문 5편, 영업 연습 3회 정도만 받기

**2) 3개월 후: 부분 유료 전환**

고객에게 이렇게 말하세요:

```
그동안 무료로 서비스를 제공해드렸는데요,
이제 정식 서비스를 오픈합니다.

무료 플랜: 월 2회 이용 가능
프리미엄 플랜: 월 3만원, 횟수 무제한 + 더 자세한 분석

지금 가입하시면 첫 달 50% 할인해드립니다!
```

**3) 예상 수익**
- 고객 10명 × 월 3만원 = 월 30만원
- 고객 30명 × 월 3만원 = 월 90만원
- 하루 1~2시간만 투자하면 됩니다

## 8. 자주 묻는 질문

### Q: Claude Pro가 뭐예요? 꼭 사야 하나요?
A: 네. AI 쓰려면 필요합니다. 월 3만원이에요. 이거 하나면 모든 AI 직원을 쓸 수 있어요.

### Q: git이 뭐예요? 설치해야 하나요?
A: AI 직원 파일을 다운로드하는 도구예요.
https://git-scm.com/download/win 에서 다운로드 → 설치 → 다음다음다음

### Q: 명령 프롬프트(까만 창)가 뭔가요?
A: 컴퓨터에 명령을 내리는 창이에요.
`윈도우키 + R` → `cmd` 입력 → 엔터

### Q: 에이전트(@무엇무엇)가 안 먹혀요
A: Claude Code 실행한 폴더에 AI 직원 파일이 없어서 그래요.
2번 단계에서 파일 복사한 걸 다시 확인해보세요.

### Q: 영어로 답변이 와요
A: 프롬프트 맨 앞에 "한국어로 답변해주세요" 라고 한 줄 추가하세요.

### Q: 기술을 전혀 모르는데 할 수 있을까요?
A: 네. 이 가이드에 나온 대로 복사+붙여넣기만 하면 됩니다.
코딩 지식 전혀 필요 없어요.

## 10. 마지막 한마디

이 서비스의 핵심은 간단합니다:

1. Claude Code라는 AI 비서를 설치한다 (10분)
