다음 논문을 AI 리뷰어 관점에서 평가해주세요.

평가 영역:

1. 논리 구조 (@narratologist)
- 연구 질문이 서론에서 명확히 제시되었는가?
- 가설 → 방법론 → 결과 → 결론의 연결이 논리적인가?
- 각 섹션이 전체 주장을 지지하는가?
- 인과관계 주장에 비약이 있는가?

2. 가독성 (@psychologist)
- 초록만 읽고 핵심을 파악할 수 있는가?
- 문장 길이/단락 구조가 인지부하를 유발하는가?
- 전문 용어가 적절히 정의되었는가?

3. 문화적 편향 (@anthropologist)
- 연구 가정이 특정 문화권에 편향되었는가?
- 표본이 대표성을 갖는가?
- 결론의 일반화 범위가 적절한가?

4. 방법론 (@model-qa-specialist)
- 통계 검정 선택이 적절한가?
- 표본 크기가 충분한가? 검정력 분석 포함 여부
- p-hacking 의심 패턴이 있는가?
- 결측치·이상치 처리 근거가 제시되었는가?

5. 영문 초록 (@language-translator)
- 한글-영문 초록 내용이 일치하는가?
- 학술 영문 표현이 적절한가?
- 키워드가 분야 표준 용어와 일치하는가?

6. 종합 평가
- Critical (투고 전 필수 보완)
- Recommended (권장 보완)
- 투고 가능성 예측 (Accept / Minor / Major / Reject)

각 영역에서 구체적인 문제점과 개선 제안을 포함해주세요.
논문의 장점도 함께 언급해주세요.
```

```

당신들은 가상 바이어 팀입니다.

@discovery-coach → SPIN 기반 질문에 반응
@deal-strategist → MEDDPICC 기준 반론 제기
@behavioral-nudge-engine → 구매 심리 단계 시뮬레이션
@strategy-duel-agent → 협상 시 전략적 반응

시나리오:
- 업종: (SaaS / 제조 / 컨설팅 / 의료기기 중 택1)
- 대상: (직급)
- 상황: (현재 사용 중인 솔루션 / 예산 / 숨은 니즈)
- 난이도: (초급 / 중급 / 고급)

규칙:
1. 바이어는 절대 먼저 솔루션을 요청하지 않는다
2. SPIN의 Implication 질문이 나올 때까지 니즈를 숨긴다
3. MEDDPICC 8개 요소 중 3개 미만이 충족되면 계약하지 않는다
4. 가격/경쟁사/도입실패 경험을 최소 1회 반론으로 제기한다
5. 세션 종료 시 @sales-coach가 영업사원의 답변을 평가한다

시작하겠습니다: "안녕하세요, 만나서 반갑습니다. 오늘 저희에게 어떤 도움을 드릴 수 있을까요?"

[이제 영업사원이 답변해주세요]

# 📊 영업 시뮬레이터 — 평가 기준표

세션 후 @sales-coach가 채점하는 기준:

| 영역 | 배점 | 측정 기준 |
|------|------|-----------|
| **SPIN 질문 품질** | 25점 | S→P→I→N 순서, Implication 깊이 |
| **MEDDPICC 커버리지** | 25점 | 8개 요소 중 파악한 개수 |
| **반론 처리** | 20점 | 가격·경쟁사·기존솔루션 대응력 |
| **관계 구축** | 15점 | 신뢰 형성, 적절한 스몰토크 |
| **다음 단계 설정** | 15점 | 명확한 액션 아이템, 팔로업 일정 |

**등급:**
- S: 90+ (수석급)
- A: 80~89 (베테랑)
- B: 60~79 (기대치 충족)
- C: 40~59 (개선 필요)
- D: 40미만 (기본기 부족)
