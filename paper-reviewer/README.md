# 📜 AI 논문 리뷰어 — 무료 서비스 빌드

## 💡 왜 이걸 만드는가?

대학원생과 연구자들은 논문 투고 전에 지도교수나 동료에게 리뷰를 부탁한다.
하지만:
- 교수님은 바쁘다
- 동료는 같은 분야가 아닐 수 있다
- 영문 교정 비용은 편당 30~50만원
- 방법론 오류는 나중에 리젝 사유가 된다

**AI 논문 리뷰어는 투고 전 1차 검증을 무료로 해준다.**
경쟁자? 없다. 논문 교정 서비스는 있지만 "구조+방법론+통계+논리"를 함께 봐주는 AI는 없다.

## 📋 리뷰 리포트 구성 (6개 영역)

### 1. 📜 논리 구조 분석 (Narratologist)
- 연구 질문이 명확한가?
- 가설 → 방법론 → 결론의 연결이 논리적인가?
- 각 섹션이 전체 주장을 지지하는가?
- 인과관계 주장에 비약이 없는가?

### 2. 🧠 가독성 평가 (Psychologist)
- 초록만 읽고 핵심 메시지를 파악할 수 있는가?
- 문장 길이와 단락 구조가 인지부하를 유발하지 않는가?
- 전문 용어가 적절히 정의되었는가?
- 독자가 길을 잃을 만한 구간은 어디인가?

### 3. 🌍 문화적 맥락 검증 (Anthropologist)
- 연구 가정이 특정 문화권에 편향되어 있지 않은가?
- 표본이 서구/남성/대학생에 과도하게 치우쳐 있지 않은가?
- 결론의 일반화 범위가 적절한가?

### 4. 🔬 방법론 검증 (Model QA Specialist)
- 통계 검정 선택이 적절한가? (t-test vs ANOVA vs non-parametric)
- 표본 크기가 충분한가? 검정력 분석 포함 여부
- p-hacking 의심 패턴: 다중 비교 보정 적용 여부
- 결측치 처리 방법이 명시되어 있는가?
- 이상치 처리 근거가 제시되어 있는가?

### 5. 🌐 영문 초록 검토 (Language Translator)
- 한글 초록과 영문 초록이 일치하는가?
- 학술 영문 표현이 적절한가?
- 키워드 선정이 해당 분야의 표준 용어와 일치하는가?

### 6. 📊 종합 평가 (Aggregate)
- 투고 전 필수 보완 사항 (Critical)
- 투고 전 권장 보완 사항 (Recommended)
- 투고 가능성 예측 (Accept / Minor Revision / Major Revision / Reject)

## 🖥️ Claude Code 명령어셋 — 논문 리뷰

### 1단계: 논문 업로드 및 분석 지시
```
@narratologist 이 논문의 논리 구조를 분석해줘.
연구 질문 → 가설 → 방법론 → 결과 → 결론의 연결성을 평가하고,
비약이 있거나 논리가 빈약한 부분을 지적해줘.

[논문 파일 첨부 or 전문 복사]
```

### 2단계: 병렬 분석 (동시에 4개 요청)
```
@psychologist 이 논문을 처음 읽는 독자 입장에서 가독성을 평가해줘.
인지부하가 높은 구간, 전문 용어 남용, 문장 구조 문제를 지적해줘.

@anthropologist 이 논문의 연구 가정과 표본에 문화적 편향이 있는지 검토해줘.
결론의 일반화 범위가 과도하지 않은지 평가해줘.

@model-qa-specialist 이 논문의 통계 방법론을 검증해줘.
검정 선택, 표본 크기, p-hacking 의심 패턴, 결측치 처리, 이상치 처리 근거를 중점적으로 봐줘.

@language-translator 이 논문의 한글 초록과 영문 초록을 비교 검토해줘.
내용 일치 여부, 학술 영문 표현의 적절성, 키워드 표준성을 평가해줘.
```

### 3단계: 종합 리포트 생성
```
@document-generator 위 5개 분석 결과를 종합해서
"AI 논문 리뷰 리포트" PDF를 생성해줘.

포맷:
- 표지: 논문 제목, 저자, 리뷰 일자
- 영역별 평가 (5개 섹션)
- 종합 평가: Critical / Recommended
- 투고 가능성 예측

전문적이고 학술적인 톤으로.
```

## 💰 무료 서비스로 운영하는 방법

### 1단계: 대학원 커뮤니티에 진입
- **김박사넷** (phdkim.net) — 국내 대학원생 커뮤니티 1위
- **하이브레인넷** (hibrain.net) — 연구자 커뮤니티
- 각 학교 대학원 총학생회 단톡방/밴드
- 페이스북 그룹 "대학원생 대나무숲"

### 2단계: 무료 서비스 제공 방식
```
제목: [무료] 논문 투고 전 AI 사전 리뷰 해드립니다

안녕하세요, AI 논문 리뷰어 서비스를 무료로 운영 중입니다.

국내 최초로 AI가 논문의 6가지 영역을 종합 분석해드립니다:
✅ 논리 구조 (주장 전개, 논증 타당성)
✅ 가독성 (인지부하, 문장 구조)
✅ 문화적 편향 (표본 대표성, 일반화 범위)
✅ 방법론 (통계 검정, p-hacking, 결측치)
✅ 영문 초록 (번역 품질, 학술 표현)
✅ 종합 리포트 (투고 가능성 예측)

신청 방법: 아래 구글폼으로 논문 제출
→ 24시간 내 PDF 리포트 이메일로 발송

[구글폼 링크]

현재 베타 테스트 중이며, 올해 12월까지 완전 무료입니다.
선착순 주 20편만 받습니다.
```

### 3단계: 구글폼 구성
```
1. 이름/소속/이메일
2. 논문 제목
3. 학술 분야 (드롭다운)
4. 투고 예정 저널 (선택)
5. 논문 파일 업로드 (PDF/Word)
6. 특별히 신경 쓰이는 부분 (선택)
```

## 📊 무료로 시작하는 이유 — 후속 수익화 전략

| 단계 | 기간 | 전략 |
|------|------|------|
| 베타 | 1~3개월 | 완전 무료, 주 20편, 입소문 |
| 성장 | 3~6개월 | 무료(주 5편) + 프리미엄(주 30편, 월 3만원) |
| 확장 | 6~12개월 | 저널별 맞춤 리뷰(월 5만원), 영문 교정 연계(건당 10만원) |

**핵심:** 무료로 쌓은 사용자 1,000명 → 프리미엄 전환율 3% = 30명 × 월 3만원 = 월 90만원  
플러스 영문 교정 연계 수수료.
