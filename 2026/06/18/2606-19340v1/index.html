Zero-shot 장기 촉수 조작: 다중 뷰 3D VLM 추론

📄 2026-06-18 로봇 arXiv: 2606.19340v1
한 줄 요약: calibrated 다중 뷰 RGB 이미지로 언어 명령을 3D 작업 계획으로 변환하는 zero-shot 프레임워크,未见物体과 새로운 장면에서도 실행 가능

authors: Jisoo Kim, Sangwon Baik, Taeksoo Kim, Sungjoo Kim, Junyoung Lee, Mingi Choi, Hanbyul Joo

Zero-shot 장기 촉수 조작을 위한 프레임워크로 calibrated 다중 뷰 RGB 이미지만으로 언어 명령을 실행 가능한 3D 작업 계획으로 변환한다. End-to-end 정책을 학습하는 대신 VLM으로 2D keypoint를 생성하고 다중 뷰 fusion으로 3D로 lift하여 triangulation과 reference-view ray voting을 결합한다. Pick-and-place와 도구 사용 모두 지원하며 실제 실험에서 단일 뷰 RGB-D grounding 및 파인 튜닝된 VLA baseline 대비 향상된 3D grounding 정확도와 실행 신뢰성을 보였다. 폐쇄 루프 상태 확인 및 재계획을 통해未见物体과 새로운 장면의 도구 사용 태스크에서 zero-shot 실행을 시연했다.
